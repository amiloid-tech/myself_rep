Документация по математическим выражениям

LaTeX формат

Элемент LaTeX синтаксис Пример
Дробь \frac{числитель}{знаменатель} \frac{a+b}{c+d}
Квадратный корень \sqrt{выражение} \sqrt{x^2 + 1}
Степень основание^{показатель} x^{2}
Индекс основание_{индекс} x_{1}
Интеграл \int_{низ}^{верх} \int_{0}^{1} x dx
Сумма \sum_{низ}^{верх} \sum_{n=1}^{\infty}
Греческие буквы \alpha, \beta, \gamma \alpha + \beta = \gamma
Тригонометрия \sin, \cos, \tan \sin(\pi)

Простой формат (автоконвертация)

Элемент Простой синтаксис Преобразуется в
Дробь (a+b)/(c+d) \frac{a+b}{c+d}
Умножение a * b a \cdot b
Степень x^2 x^{2}
Индекс x_1 x_{1}

Советы по использованию

  • Используйте кнопку "Автоконвертация" для преобразования простого формата в LaTeX
  • Для сложных выражений рекомендуется использовать LaTeX синтаксис
  • Вы можете скачать результат как изображение PNG
  • Кликните на примеры выражений для быстрой вставки
  • Нажимайте Enter для преобразования выражения
